# converter Uploader

Программа для загрузки файлов на converter с генерацией PowerShell команды.

## Варианты использования

### 1. Веб-версия (index.html)

Просто открой `index.html` в браузере:
- Перетащи файл в область загрузки или кликни для выбора
- Нажми "Загрузить на converter"
- Получи готовую PowerShell команду
- Кликни на команду чтобы скопировать

### 2. Python-версия (uploader.py)

**Требования:**
```bash
pip install requests
```

**Использование:**
```bash
python uploader.py путь_к_файлу
```

Или просто перетащи файл на `uploader.py` в проводнике.

## Как это работает

1. Файл загружается на converter (временное хранилище Litterbox)
2. Генерируется PowerShell команда: `irm <url> | iex`
   - `irm` (Invoke-RestMethod) - скачивает файл
   - `iex` (Invoke-Expression) - выполняет скачанный скрипт

## Внимание!

PowerShell команда `irm ... | iex` скачивает и **сразу выполняет** файл. 
Используй только с доверенными файлами!
