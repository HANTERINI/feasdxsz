# Как запустить crypter Uploader на Vercel

Я подготовил файлы для деплоя на Vercel. 

### Важные ограничения Vercel (Free tier):
1. **Размер файла:** Лимит на загрузку файла — **4.5 МБ**. Если файл больше, сервер выдаст ошибку. Для больших файлов используйте локальный сервер.
2. **Локальный конвертер:** Кнопка скачивания конвертера **не будет работать** на Vercel, так как у облака нет доступа к файлам на вашем рабочем столе.

---

### Инструкция по деплою:

#### Вариант 1: Через GitHub (Рекомендуется)
1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из этой папки в репозиторий.
3. Зайдите на [vercel.com](https://vercel.com) и создайте новый проект.
4. Выберите ваш репозиторий и нажмите **Deploy**.

#### Вариант 2: Через Vercel CLI
1. Установите Vercel CLI: `npm i -g vercel`
2. Откройте терминал в папке проекта.
3. Введите команду: `vercel`
4. Следуйте инструкциям (просто жмите Enter).

---

### Файлы, созданные для деплоя:
- `package.json` — настройки проекта и зависимостей.
- `vercel.json` — конфигурация маршрутов (маршрут `/upload` теперь ведет на серверную функцию в облаке).
- `api/upload.js` — серверная логика загрузки для Vercel.
- `api/download-converter.js` — заглушка для скачивания (на Vercel не работает с локальными путями).
